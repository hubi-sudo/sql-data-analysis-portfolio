### 📊 SQL Data Analysis Portfolio

Welcome to my data analysis portfolio. This repository contains a collection of my SQL projects, showcasing my ability to extract, manipulate, and analyze large datasets using relational databases (MySQL, SQLite).

## 🏆 Major Projects

### 1. Spotify Dataset - Exploratory Data Analysis (EDA)
Comprehensive exploratory data analysis performed on a large-scale Spotify dataset. 
* **Key Skills Applied:** Data cleaning, type casting, advanced window functions (`ROW_NUMBER`), and multi-level aggregation (`GROUP BY`, `HAVING`).
* **Objective:** Identifying historical trends, ranking tracks, and extracting business-valuable metrics from raw music data.

### 2. Corporate Employees Database - Salary Grid Analytics
Complex multi-table analysis executed on the official MySQL Employees database (over 3 million records).
* **Key Skills Applied:** Cascading Common Table Expressions (CTEs), multi-table `JOIN` operations, and data aggregation.
* **Objective:** Consolidating data from `employees`, `salaries`, and `titles` tables to calculate and analyze average salaries across different corporate roles, filtering out statistically insignificant data.

## 📌 Ad-hoc Queries & Explorations
Alongside the major reports, this repository includes smaller, targeted scripts used for specific data extraction tasks:
* **Top Earners Analysis:** Quick-extraction scripts using `ORDER BY` and `LIMIT` to identify historical top salaries.
* **Basic Data Profiling:** Initial database exploration and schema validation queries.

## 🛠️ Technology Stack
* **Languages:** SQL (MySQL, SQLite), Python (Basic data manipulation)
* **Tools:** MySQL Workbench, DB Browser for SQLite, Git
